# Visual Studio Dark Modern Neue

A refined Dark Modern theme family for Visual Studio Code, built on the Dark 2026 palette with custom warm tones and semantic color hierarchies.

## Variants

### Dark Modern Neue
The base variant. Warm desaturated tones inspired by Nord — salmon strings, teal types, sky variables. Chromatic balance over high contrast.

### Dark Modern Neue (High Contrast)
High saturation Tailwind-based palette. Electric indigo, fuchsia, sky, and emerald with strong chromatic separation. Built for maximum token differentiation.

### Dark Modern Neue (Minimal)
Monochrome with a single teal accent (`#7DCFCA`) for strings. A clean, distraction-free coding environment built on a neutral gray scale.

## Color Philosophy

**Neue (Default)** — Desaturated warm palette. Keywords recede, identifiers breathe. Luminosity-matched across token families.

**Neue High Contrast** — Full saturation Tailwind colors. Every token family has a distinct hue. Built for long sessions where clarity is critical.

**Neue Minimal** — Near-monochrome neutral scale. The teal accent is the only chromatic element. Keywords are darker than identifiers — the code reads through content, not syntax.

## Installation

1. Open VS Code
2. Go to Extensions (`Ctrl+Shift+X`)
3. Search for `Dark Modern Neue`
4. Click Install
5. Open Command Palette (`Ctrl+Shift+P`) → `Preferences: Color Theme`
6. Select your preferred variant

## Recommended Settings

```json
{
  "editor.fontFamily": "CommitMono",
  "editor.fontSize": 13,
  "editor.lineHeight": 1.5,
  "editor.letterSpacing": 0.15,
  "editor.fontLigatures": true,
  "editor.fontVariations": true,
}
```
